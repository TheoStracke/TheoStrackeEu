# Logos das Empresas e Instituições

Coloque os arquivos de logo nesta pasta com os seguintes nomes:

## Experiência Profissional:
- `vellum.png` - Logo da Rede Vellum
- `thomson-reuters.png` - Logo da Thomson Reuters Brasil
- `exercito.png` - Logo do Exército Brasileiro

## Educação:
- `senac.png` - Logo do Senac SC

## Requisitos dos arquivos:
- Formato: PNG (com fundo transparente, preferencialmente)
- Tamanho recomendado: 200x200px a 400x400px
- Os logos serão redimensionados automaticamente para caber em círculos de 64x64px
- Use logos oficiais em alta qualidade

## Onde encontrar:
- Sites oficiais (seção de imprensa/mídia geralmente tem logos para download)
- Google Images (pesquise por "[nome da empresa] logo png")
- Para o Senac: https://www.sc.senac.br/
